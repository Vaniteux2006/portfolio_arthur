# Projeto de Conclusão de Curso - Portfólio Pessoal

Este repositório contém os arquivos relacionados ao meu projeto de conclusão de curso, que visa a criação de um portfólio profissional.
Gostaria de ressaltar que este projeto ainda está em desenvolvimento e, portanto, pode conter funcionalidades incompletas ou ausentes.

# Objetivo
O objetivo deste projeto é criar um portfólio interativo sobre mim mesmo. Através dele, pretendo utilizar futuramente para uso próprio, seja como parte de meu currículo profissional ou como demonstração do que posso fazer.

# Status do Projeto
Atualmente, o projeto encontra-se em fase de desenvolvimento ativo. Embora já tenha sido implementada uma parte significativa do código, ainda há muito trabalho a ser feito. Algumas das funcionalidades-chave planejadas incluem:

- TROCA DE PALHETA DE CORES;
- ACESSO A PROJETOS PRÓPRIOS;
- INTERAÇÃO COM A FOTO;
- ACESSIBILIDADE A DISPOSITIVEIS MÓVEIS.

# Contribuições
Aceito contribuições para este projeto. Se você estiver interessado em colaborar, sinta-se à vontade para fazer um fork deste repositório, fazer as alterações necessárias e enviar um pull request. Ficarei feliz em revisar suas contribuições e considerá-las para inclusão no projeto.
